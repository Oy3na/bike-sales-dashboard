📊 Overview
This project analyzes customer data to identify factors influencing bike purchases. The dashboard provides insights into income levels, commute distance, and age groups.

🎯 Objectives
Understand customer demographics
Identify trends in bike purchasing behavior
Support data-driven marketing decisions

📈 Key Insights
Higher-income individuals are more likely to purchase bikes
Customers with shorter commute distances show higher purchase rates
Middle-aged individuals are the primary buyers

🛠️ Tools Used
Microsoft Excel (Dashboard & Data Analysis)
📷 Dashboard Preview

📊 Workbook Structure
The Excel workbook contains multiple sheets used during the data analysis process.
